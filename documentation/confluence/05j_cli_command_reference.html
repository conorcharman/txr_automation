Entry pointaccuracy_testing.scripts.buyer_id_validation:main
PurposeValidate buyer identification codes (NIDN, CONCAT, CCPT) for incidents 7_35, 7_37, 7_39

Common Arguments

Most CLI scripts share these common arguments:

ArgumentDescriptionDefault
--config PATHPath to YAML configuration fileAuto-discovers config/local/...
--use-envUse TXR_* environment variables instead of YAML filefalse
--log-level LEVELOverride logging level (DEBUG, INFO, WARNING, ERROR)INFO
--dry-runPreview validation without writing output filesfalse
--progressShow progress bar during processingfalse
--verboseEnable verbose console outputfalse
--helpShow help text and available arguments—
