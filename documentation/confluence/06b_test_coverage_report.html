Running the Test Suite

Full Suite

conda activate txr_automation
pytest tests/ -v

By Module

pytest tests/test_accuracy_testing/ -v
pytest tests/test_core/ -v
pytest tests/test_replay/ -v
pytest tests/test_firds/ -v
pytest tests/test_gleif/ -v
pytest tests/test_utils/ -v
pytest tests/integration/ -v

With Coverage

pytest tests/ --cov=src --cov-report=html --cov-report=term-missing

Specific Test Pattern

pytest tests/ -k "buyer" -v          # All buyer-related tests
pytest tests/ -k "concat" -v         # All CONCAT tests
pytest tests/ -k "not integration"   # Exclude integration tests
CI/CD: The full test suite runs on every commit via scripts/run_tests.sh. Coverage reports are generated with scripts/run_tests_with_coverage.sh.
