# Check if a UK NINO matches expected format
id_format_manager.validate_format("GB", "NIDN", "AB123456C")  # True

# Get detailed mismatch reason
pattern = id_format_manager.get_pattern("GB", "NIDN")
pattern.get_mismatch_reason("INVALID")  # "Expected 9 characters, got 7"

safe_open_csv()

def safe_open_csv(file_path: Path, mode: str = 'r', **kwargs) -> Tuple[TextIO, str]

Opens a CSV file with automatic encoding detection (UTF-8 → UTF-8-sig → Latin-1 fallback). Returns (file_handle, encoding_used).

CharacterReplacement

Handles special character encoding issues (Latin-1 ↔ UTF-8 byte-level corrections) common in AS/400 export data.

Data Structures

ReplayRecord (dataclass)

Represents a transaction record in the replay processing pipeline. Contains transaction reference, client details, ID codes, and matching metadata.

LookupResult (dataclass)

Result of a transaction reference lookup, including match status, matched row data, and confidence score.

ProcessingStats (dataclass)

Aggregated processing statistics with record counts, match rates, error counts, and timing data. Implements to_dict() for structured logging.
