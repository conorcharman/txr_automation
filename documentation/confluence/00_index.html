TXR Automation Documentation

Transaction Reporting Automation Suite — A modern full-stack web application for validating financial transaction data against UK MiFIR/RTS 22 regulatory standards. Built with React 19, FastAPI, PostgreSQL, and Redis. This documentation covers the web application architecture, API reference, deployment procedures, core validation engine, and historical VBA migration context.
